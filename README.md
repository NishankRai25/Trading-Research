This repository contains my research, trading code, and development process as I learn to build quantitative strategies.

The focus is not on generating alpha yet, but on developing strong fundamentals: reading research papers, implementing models from scratch, and making small, targeted improvements to understand how they work in practice.

Each strategy in this repository is deployed through Interactive Brokers (IBKR) or Alpaca using paper trading. The goal is to simulate real-world conditions while building intuition for model behavior, execution, and risk.

This is a learning-focused environment designed to improve my ability to translate theory into working trading systems.
